tag de hiperlink

tag a href de fine um hiperlink (conexão) que é utilizada para vincular uma página a outro.

o atributo mais importante desta a ytag é o href que define a página de destino que será exibida após click do mouse no link.

Por padrão, os links aparecerão da seguinte dorma em todos os naveguadores:

Um link não vistado é sublinado de azul

Um link não vistado é sublinado de azul

um link vistado é sublinado de roxo.

um link ativo é sublinado de vermelho
